To get your JWT token:\n1. Open browser DevTools (F12)\n2. Go to Application tab → Local Storage → http://localhost:3000\n3. Copy value of "token" key\n4. Edit cleanup-patients.sh - replace YOUR_JWT_TOKEN_HERE\n5. Run: chmod +x cleanup-patients.sh && ./cleanup-patients.sh\n\nOr use browser console:\nfetch("http://localhost:5000/api/patients/P001", { method: "DELETE", headers: {"Authorization": "Bearer " + localStorage.getToken() } })
